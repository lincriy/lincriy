第四模块 HIVE搭建

发表于2023-04-01|更新于2023-04-17
|字数总计:573|阅读时长:2分钟

HIVE数仓(mysql slave2)

1.关闭mysql开机自启

systemctl disable mysqld

2.开启mysql服务

systemctl start mysqld

3.判断mysqld.log下是否生成出临时密码

grep password /var/log/mysqld.log

4.设置mysql数据库本地root用户密码为123456

mysql -uroot -p
> 此处写初始密码
alter user root@'%' identified by '123456';

5.解压hive安装包至指定目录

tar -zxf /root/package277/apache-hive-2.3.4-bin.tar.gz -C /usr/hive

6.配置hive环境变量，并生效

vim /etc/profile
export HIVE_HOME=/usr/hive/apache-hive-2.3.4-bin
export PATH=$PATH:$HIVE_HOME/bin
. /etc/profile

7.设置hive运行环境，配置Hadoop_home,hive_conf_dir（hive配置文件）,hive_aux_jars_path(资源库)

vim /usr/hive/apache-hive-2.3.4-bin/conf/hive-env.sh
export HADOOP_HOME=/usr/hadoop/hadoop-2.7.7
export HIVE_CONF_DIR=/usr/hive/apache-hive-2.3.4-bin/conf
export HIVE_AUX_JARS_PATH=/usr/hive/apache-hive-2.3.4-bin/lib

8.解决jline冲突

cp $HIVE_HOME/lib/jline-2.12.jar $HADOOP_HOME/share/hadoop/yarn/lib
slave1作为服务器配置hive-site.xml

9.拷贝jdbc驱动到对应的lib下

cp mysql-connector-java-5.1.47-bin.jar $HIVE_HOME/lib

10.配置元数据存储位置，数据库连接，连接jdbc的URL地址主机名及默认端口号3306，自行创建数据库，ssl连接为false,数据库连接用户/密码

vim $HIVE_HOME/conf/hive-site.xml

<configuration>
<property>
<name>hive.metastore.warehouse.dir</name>
<value>/usr/hive_remote/warehouse</value>
</property>
<property>
<name>javax.jdo.option.ConnectionDriverName</name>
<value>com.mysql.jdbc.Driver</value>
</property>
<property>
<name>javax.jdo.option.ConnectionURL</name>
<value>jdbc:mysql://slave2:3306/hive?createDatabaseIfNotExist=true&amp;useSSL=false</value>
</property>
<property>
<name>javax.jdo.option.ConnectionUserName</name>
<value>root</value>
</property>
<property>
<name>javax.jdo.option.ConnectionPassword</name>
<value>123456</value>
</property>
</configuration>
master作为客户端的配置hive-site.xml
vim $HIVE_HOME/conf/hive-site.xml

<configuration>
<property>
<name>hive.metastore.uris</name>         #此处的uris指向服务器（slave1）开放的端口
<value>thrift://slave1:9083</value>
</property>
<property>
<name>hive.metastore.local</name>         #是否关闭本地模式
<value>true</value>
</property>
<property>
<name>hive.metastore.warehouse.dir</name>     #配置客户端hive元数据存放的位置
<value>/usr/hive_remote/warehouse</value>
</property>
</configuration>
hive服务器初始化数据库，启动metastore服务(slave1)
schematool --dbType mysql --initSchema         #初始化hive,mysql出现57张表
hive --service metastore  #启动hive服务
hive客户端启动hive，并创建hive数据库（master）
hive
create database hive;
