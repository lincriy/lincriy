mysql安装整合版

发表于2023-04-01|更新于2023-04-17
|字数总计:130|阅读时长:1分钟

5.0安装好之后一共6条命令：

1.   mysqld --initialize-insecure --user=mysql
2.   systemctl start mysqld
3.   mysql
4.   grant all privileges on *.*  to root@'%' identified by '比赛要求改成的远程登录密码';
5.   exit或Ctrl+d
6.   mysqladmin password '比赛要求改成的本地密码'

8.0安装好之后一共7条命令：

1.   mysqld --initialize-insecure --user=mysql
2.   systemctl start mysqld
3.   mysql
4.   CREATE USER root@'%' IDENTIFIED BY '比赛要求改成的远程登录密码';
5.   grant all privileges on *.* to root@'%';
6.   exit或Ctrl+d
7.   mysqladmin password '比赛要求改成的密码'
